# Catchy Decors website

Open `index.html` in any modern browser, or upload this entire folder to your web host.

Before publishing, replace the `loc` values in `sitemap.xml` with your final full website address (for example, `https://your-domain.com/index.html`). This lets search engines index the site correctly.

The contact form is a front-end enquiry form. Connect it to your preferred email/form service before publishing if you would like submissions delivered automatically.
